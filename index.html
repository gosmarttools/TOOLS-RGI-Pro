TOOLS RGI Pro - README.md Generator Interactive

Drag & drop logo atau klik kotak untuk memilih file. Isi semua field → klik Generate README → Download README.md

Panduan Penggunaan Tool:

  • Isi semua field form dengan informasi proyek Anda.
  • Gunakan @username untuk owner dan team GitHub.
  • Field license gunakan format: MIT - https://opensource.org/licenses/MIT
  • Live Project URL bersifat opsional, jika ingin menampilkan placeholder screenshot.
  • Setelah klik Generate, preview Markdown & HTML akan muncul di bawah form.
  • Download README.md untuk langsung upload ke repo GitHub Anda.
