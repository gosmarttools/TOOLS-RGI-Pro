## Cara Pakai

1. Buka **index.html** melalui GitHub Pages atau langsung di browser.
2. **Drag & drop logo** atau klik kotak logo untuk memilih file.
3. Isi semua field form:
   - Judul Repo
   - Star / Fork Count
   - License
   - About / Philosophy
   - Architecture Overview
   - Modules
   - Tech Stack
   - Local Development Instructions
   - Owner GitHub (@username)
   - Team Members (@username)
   - Contribution Guide
   - Contributors
   - Roadmap
   - Live Project URL (opsional)
   - Technical Blog & Updates URL
4. Klik **Generate README** untuk melihat preview Markdown & HTML.
5. Klik **Download README.md** untuk menyimpan file siap upload ke repo GitHub.

## Fitur Utama
- Drag & drop logo
- Ambil avatar GitHub otomatis
- Preview Markdown & HTML langsung
- Screenshot live project otomatis (opsional, perlu API key)
- Download file `.md` siap pakai

## Catatan
- Jika tidak ingin menggunakan screenshot live project, bisa kosongkan field URL.
- Pastikan memasukkan GitHub username dengan format `@username`.
- Logo default tersedia di folder `assets/logo-default.png`.
